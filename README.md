# security-learning

Media belajar cyber security dari nol, dengan target sertifikasi **CompTIA Security+ (SY0-701)**.

## Tentang repo ini

Repo ini berisi catatan materi dan latihan soal yang saya susun selama proses belajar, dari fundamental sampai siap ujian sertifikasi. Materinya dibahas per topik lewat diskusi interaktif (tanya-jawab, koreksi kesalahan, mnemonic), baru dicatat rapi di sini untuk direview ulang nanti.

## Metode belajar

1. Materi disampaikan per topik lewat diskusi, lengkap dengan analogi dan mnemonic biar gampang diingat
2. Tiap topik ditutup dengan quick check (2-4 soal) buat verifikasi pemahaman
3. Kesalahan langsung dikoreksi dengan penjelasan detail, terutama konsep yang mirip-mirip dan sering ketuker
4. Materi final dicatat di repo ini untuk direview ulang menjelang ujian
